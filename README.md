# Patriot

Top-down 2D multiplayer co-op action game.

## Setup

```bash
npm install
npm run build --workspace=@patriot/shared
```

## Development

Run both server and client:

```bash
npm run dev
```

Or run individually:

```bash
npm run dev:server   # http://localhost:2567
npm run dev:client   # http://localhost:5173
```

## Mobile Testing

Connect your phone to the same WiFi and open `http://<your-PC-IP>:5173`.
